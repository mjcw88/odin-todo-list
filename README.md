# The Odin Project (Full Stack Javascript) - Todo List
Todo List project for the Odin Project Full Stack Javascript course. Featuring extensive use of IIFE, objects, factory functions, DOM manipulation, localStorage, module import/exports. In place of a database, localStorage was used to save and load data as databases is yet to be covered in the course.

One feature that is missing that would be worth considering in terms of scalability is paginating the tasks/todo's but for the scope of this project it is not necessary.

Full project specs can be found [here](https://www.theodinproject.com/lessons/node-path-javascript-todo-list).
